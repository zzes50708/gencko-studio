## 3. 視覺系統

### 配色（黑底 + 品牌橘高光）

- 背景 `#07080a`｜霧 `#0b0d12`｜DNA 守宮主體橘 `#ee6a2e`（呼應品牌 `#E8440A`）｜bloom 芯 `#ff8a4c`｜冷輔助 `#4a6b8a`｜主文字 `#f2f0ec`｜次文字 `#9aa0a6`

### 後製堆疊（EffectComposer 順序）

1. RenderPass → 2. UnrealBloomPass（0.8–1.4 / 0.4 / 0.85）→ 3. DOF（bokeh，聚焦主體）→ 4. Vignette →（Full 限定）5. 極輕 film grain / chromatic aberration

### 字體與排版（不加字型）

- 中文：Noto Sans TC（已載入）｜大型拉丁/數字：Black Ops One（已載入）
- AT 感靠：字距加寬、全大寫拉丁、**殘影疊字**（主標背後放大級低透明度複本，如 AT 卡片「harmonic/ate」）

## 4. 中央主體：DNA 守宮

- 造型：可辨識**守宮輪廓**（沿用 `gecko-outline`），**身體由 DNA 雙螺旋 + 鹼基節點構成**；發光邊緣、立體厚度
- 建法：參數化生成（守宮輪廓取樣 + 螺旋數學 + 節點 InstancedMesh），沿用/改造 `genGeckoDNA` 粒子邏輯
- 持久不換形；隨 `progress` 只變參數：整體旋轉、螺旋張力、呼吸縮放、發光強度、節點活躍度、霧/bloom 密度
- 進場「粒子收斂成守宮」的招牌時刻 → 保留為**進入後的成形動畫**
- 守宮身上的 DNA 節點 = 卡片掛點（world-space anchor）

## 5. 空氣層（粒子 / 霧 / 光）

- 前景高亮粒子（Full ≤3000 / Lite ≤1200 / 手機不跑）、背景冷灰塵點、`fogExp2` 體積霧、Full 限定假光束
- 動法：shader（uTime + curl noise）或 GPGPU；Lite 退回簡單 shader 位移

## 6. 進場門與捲動模型

### 進場門（AT 式沉浸入口）

1. 近黑畫面 + logo + 載入進度（資產/場景就緒度）
2. 就緒 → 出現「進入」按鈕
3. **點擊才啟動場景**（重運算/render loop 在使用者手勢後開始 → 改善感知效能與 CWV）
4. 進入後播「粒子收斂成 DNA 守宮」成形動畫，接入 timeline 起點

### 捲動模型（Lenis + GSAP，不全劫持）

- Lenis 提供動量平滑捲動；GSAP ScrollTrigger `pin` 釘住英雄區、`scrub` 把捲動連續映射成 `timelineProgress ∈ [0,1]`
- progress 推導：鏡頭 pos/target、守宮旋轉/張力/呼吸/亮度、霧/bloom、卡片活躍度、文字進出
- **廢除** step-snap / `navigateTo` / 45° 輪播
- 跑完 5 卡 → 解除 pin → 放行到下方真實內容
- 進度點（5 點）可點擊跳 progress；任一中間位置畫面都成立

### 鏡頭路徑（圍繞 DNA 守宮、短路徑、不暈眩）

| progress | 卡          | 鏡頭                 | 主體行為                        |
| -------- | ----------- | -------------------- | ------------------------------- |
| 0.00     | 成形/待機   | 中景偏上 5–10°       | 粒子收斂成守宮，低頻呼吸        |
| ~0.15    | 01 關於     | 略前推               | 守宮成形穩定，兩入口節點微亮    |
| ~0.35    | 02 基因圖鑑 | 左前切，聚焦身體螺旋 | DNA 螺旋條帶展開                |
| ~0.55    | 03 特寵醫院 | 略拉開               | 延伸節點，冷色資料光            |
| ~0.75    | 04 計算機   | 靠近核心             | 形成運算核心 + 軌道流動         |
| ~0.92    | 05 選購     | 微抬高拉後景         | 商品節點 + 台灣輪廓（厚度景深） |

## 7. 空間卡片（AT `/work` 語彙）

### 卡片視覺（依參考圖）

- 漂浮空間面板，貼合守宮 DNA 節點 anchor（world→screen 投影），有前後層次、可被主體遮擋
- **卡面內含動態背景**（見 §10）、薄描邊/半透明底、背後殘影疊字、一顆 CTA 按鈕
- 同時只突出 1 張主卡；浮出 0.35–0.55s（先淡入再微位移），離場先降透明再輕縮

### 卡片內容與導流（點 CTA 直接跳站）

| 卡  | 主題                           | 文案要點                               | CTA → route                  |
| --- | ------------------------------ | -------------------------------------- | ---------------------------- |
| 01  | 關於 Gencko                    | 整合入口、品牌與知識                   | `前往了解` → `/about`        |
| 02  | 基因圖鑑                       | morph/表現型/品系脈絡                  | `查看基因圖鑑` → `/genes`    |

### 資料結構（文案與 3D 分離）

```ts
interface HeroCard {
  id: string
  progress: number // timeline 位置 0~1
  title: string // 主標
  desc: string // 說明（03 特寵醫院可較長）
  cta: { label: string; to: string } // 直接跳站
  anchor3D: string // 守宮 DNA 節點掛點 id
  faceAnim?: string // 卡面 shader/貼圖 id（後續動畫用）
}
```

## 9. 手機與降級

| 級別              | 條件                                | 英雄區                                        | 說明        |
| ----------------- | ----------------------------------- | --------------------------------------------- | ----------- |
| Desktop Full      | `hover:hover`+`pointer:fine`+高效能 | 完整即時 3D + 滑鼠擾動 + 卡面 shader + 全後製 | —           |
| Desktop Lite      | 中低效能                            | 即時 3D 但砍背景粒子/強 glow/grain，DPR↓      | —           |
| Mobile            | `pointer:coarse`                    | **靜態海報 / 極短預渲染片** + 真實卡片與內容  | 不跑即時 3D |
| Reduced Motion    | `prefers-reduced-motion`            | 近靜止 + 卡片切換 + 導覽                      | —           |
| WebGL 不支援/失敗 | 能力偵測失敗                        | **同手機靜態海報降級**                        | 不 redirect |

實作：`ScrollTrigger.matchMedia` + WebGL 能力偵測 + 動態 `setPixelRatio`（Full 1.5 / Lite 1.25 / 其餘 1）。

## 10. 卡面動畫（後續工作項）

- 桌機：卡面背景用**即時 render-target**（shader/粒子渲染到貼圖），與主場景同源、隨主體連動
- 手機/降級：卡面靜態圖
- 若要 AT 那種電影感旋渦影片：改預渲染短片路線（可用 `scroll-world`/Higgsfield 生成）——列為選項，非預設

## 11. 效能策略

- 幾何/材質共用、`InstancedMesh`、粒子分級、`fogExp2`、後製按級砍 pass、`THREE.Clock` deltaTime
- 監看 `WebGLRenderer.info`；卸載 `dispose()` 幾何/材質/貼圖/composer/render target
- SSR 安全：3D 僅 client 掛載（`<ClientOnly>` / `import.meta.client`），避免 hydration 崩

## 12. SEO / 無障礙

- 英雄區隨附**隱藏文字層**（仿 AT `.GLA11y`，`clip:rect(0 0 0 0)`）：含 5 卡標題+說明+連結，餵爬蟲與螢幕閱讀器
- 保留單一可見 `h1`（含「守宮」關鍵字）
- 進場門/splash 階段先出 SSR h1，場景載入後交棒
- `/about` 本次不動；其 sitemap/文章連結維持（待後續文字頁任務再處理）
